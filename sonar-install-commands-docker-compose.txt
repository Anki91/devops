sudo apt-get update

Download the script file
wget https://raw.githubusercontent.com/awstechguide/devops/master/docker-compose.yml

Create a directory
mkdir jenkins

Move the script file
mv docker-compose.yml jenkins

Change directory
cd jenkins


See what is inside the script file
cat docker-compose.yml

move out from jenkins
cd ..



Install docker compose
sudo curl -L "https://github.com/docker/compose/releases/download/1.24.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

Change permission
sudo chmod +x /usr/local/bin/docker-compose

Check the version
docker–compose –version

Bring up docker compose
sudo docker–compose up

Check the docker process
docker ps

You can check the logs of sonarqube
docker-compose logs f sonarqube
